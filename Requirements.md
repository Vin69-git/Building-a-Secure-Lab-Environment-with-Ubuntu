### 🖥️ Secure Lab Infrastructure & Platform Prerequisite Matrix

To deploy this Linux laboratory workspace successfully and enforce absolute boundary isolation from your local operating system network, ensure your infrastructure satisfies the following baseline requirements:

* **Hypervisor Compatibility Layer:** Type-2 Hardware Virtualization Software (VMware Workstation Pro 17.x+ or Oracle VirtualBox Core).
* **Source Distribution Image Core:** Hardened Linux Guest OS Image File (Ubuntu 24.04.1 LTS 64-bit Desktop ISO).
* **Minimum Compute Resource Allocation:** * Storage Profile: At least 20 GB of free block storage space on a local target drive.
  * System Memory: 4 GB RAM assigned to prevent performance bottlenecks.
  * Processing Metrics: 2 Core CPUs dedicated to handle concurrent terminal processing loops.
